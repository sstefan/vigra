VIGRA Tutorial
==============

The source code of the tutorial can be found in the following folder:

    /docsrc

It consists of the following files:

    tutorial.dxx
    important_methods.dxx
    image_processing.dxx
    
Please consider that the file "Doxyfile.in" has to be changed to compile these files. In this 
version of "Doxyfile.in" everything is fine.

The included code and the examples can be found in:
    
    /src/examples/tutorial

All the included images are stored in

    /src/images

This folder has to be mentioned in "Doxyfile.in" as well.
